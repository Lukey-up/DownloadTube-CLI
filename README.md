# DT Downloader

Downloader de vídeos para terminal feito em Python.

## Recursos

- Download de vídeos
- Interface colorida no terminal
- Criação automática da pasta DownloadTube
- Sistema de dicas aleatórias
- Executável para Windows

## Instalação

### Windows

1. Baixe o instalador na aba Releases
2. Execute o instalador
3. Abra o CMD
4. Digite:

```bash
DT
```

## Requisitos para código fonte

- Python 3.14+
- Bibliotecas do requirements.txt

Instale com:

```bash
pip install -r requirements.txt
```

## Executar pelo código fonte

```bash
python DT.py
```

## Estrutura do projeto
