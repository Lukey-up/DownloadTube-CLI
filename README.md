# DT Downloader

Downloader de vídeos para terminal feito em Python.

## Recursos

- Download de vídeos do YouTube
- Pesquisa integrada
- Interface via terminal
- Criação automática do diretório DownloadTube

## Instalação

### Windows

1. Baixe o instalador na aba Releases
2. Execute o instalador
3. Abra o CMD
4. Digite:

```bash
DT
```

## Código fonte

Clone o repositório:

```bash
git clone SEU-LINK-AQUI
```

Entre na pasta do projeto:

```bash
cd downloadtube
```

Instale as dependências:

```bash
pip install -r requirements.txt
```

Execute o programa:

```bash
python DT.py
```
